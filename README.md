# 배포주소
https://mabeopsonyeo.github.io/test/

# 시작 가이드
- Node.js v18.17.1
- npm v9.6.7
  
### 설치
```
$ git clone https://github.com/mabeopsonyeo/test
$ cd test
```

### 실행
```
$ npm install
$ npm run start
```

### 실행 가능한 스크립트
```
$ npm run start - 로컬에서 실행합니다.
$ npm run build - 프로젝트를 빌드합니다.
$ npm run deploy - gh page로 배포합니다.
```
